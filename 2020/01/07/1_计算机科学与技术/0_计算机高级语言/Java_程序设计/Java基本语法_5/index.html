1、Java 标识符

    1. Java 程序中的命名规范
    • 变量、常量、类
      • 变量用小写字母开始
      • 常量全部都是大写字母
      • 类大写字母开始
    • 约定的 Java 命名
      • 包名package：全小写，以.隔开
      • 类名class：首字母大写，每个单词的首字母都大写
      • 接口名interface：同类名
      • 方法名：第一个单词首字母小写（通常为动词），其余单词都已首字母大写为例
      • 变量名：同方法名
      • 常量名：
        • 基本数据类型：大写以”_”隔开
        • 对象类型：大小写混合，大写开头，以大写隔开单词

2、Java 注释方法

    1. 3 种注释方式
    • 单行注释
    • 块注释
    • 文档注释

3、Java 顺序、分支、循环

    1. 分支语句(可嵌套)

if else

switch

switch后面的变量类型是整数、字符、字符串；
case后面跟常量,不能是变量;
注意break;

    1. 循环语句

for 循环

while 循环

do while 循环

    1. 特殊流程控制语句
    • break
      • 终止某个语句块的执行
      • 可以通过标签指明终止哪个语句块
      • break label2 ;
    • continue
      • 跳过某个循环语句块的一次执行，下面语句块不执行
      • 多层嵌套的循环语句块中，可以通过标签指明跳过哪一层循环
